**Migration pattern for each function:**
1. Replace `callGeminiAPI('gemini-3-pro-image-preview', ...)` with `callUnifiedAPI(IMAGE_MODEL, ...)`
2. Replace `callImagenAPI('imagen-4.0-ultra-generate-001', ...)` with `callUnifiedAPI(IMAGE_MODEL, ...)` AND change the request body format from `:predict` to `:generateContent` format

4. Replace duplicated `input.includes('base64,') ? input.split(',')[1] : input` with `extractBase64(input)`
5. Extract the image from `parsed.images[0].dataUri` instead of manual `data:${mimeType};base64,${data}` construction
